GROmaρs
A GROMACS-based toolset for the calculation and comparison of density maps from molecular dynamics simulations. Here, you will find information how to download, install, and execute the toolset.

Download
1) Clone GROMACS.git, by executing in the terminal:
git clone git://git.gromacs.org/gromacs.git

2) fetch the latest patch of GROmaρs, by signing in, via a web browser, in the site:
https://gerrit.gromacs.org/#/c/8005
Click "Download" at the right-hand side. Copy-paste and execute the entire Checkout-line command in the terminal, which for "user" should be something like:
git fetch https://user@gerrit.gromacs.org/a/gromacs refs/changes/05/8005/7 && git checkout FETCH_HEAD

2-optional) Any time you want to fetch a new patch, run (2) after reseting any previous downloaded and installed patch, by executing
git reset --hard HEAD^

Installation
3) Go to the created directory
cd gromacs

4) Install GROMACS as usual (instructions here):
mkdir build
cd build
cmake .. -DGMX_BUILD_OWN_FFTW=ON -DREGRESSIONTEST_DOWNLOAD=ON
make

After installation, the GROmaρs tools will be accessible in the "build" directory as:
./bin/gmx tide -h
./bin/gmx mapxxx -h
